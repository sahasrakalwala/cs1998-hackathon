An app that shows your personalized habits and tracks them by checking them off. You can input your habits and then track how many times you completed those habits.

Requirements:
- Multiple screens between entering your name, your list of habits, and adding the habits
- Scrollable view of the habits
- Integrated the backend API to fetch the habits

You have to run the app from the most inside Xcode project.
